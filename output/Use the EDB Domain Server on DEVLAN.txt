Use the EDB Domain Server on DEVLAN
DNS Settings:
User #71384 runs our own DNSDomain Name System server.  To use these DNSDomain Name System servers, copy the following lines to the /etc/resolv.conf file:

domain edb.devlan.netsearch edb.devlan.net devlan.netnameserver 10.6.5.51nameserver 10.6.5.50nameserver 10.3.1.10
