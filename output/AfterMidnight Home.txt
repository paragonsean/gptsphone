
                        Packrat Documentation Home

Packrat is an automated framework for Listening Post creation/provisioning, configuration, and maintenance.  Originally created as a NODNetwork Operations Division iTime project, it allows users to quickly spin up LPs with several different builder options (VMWare Workstation, VMWare ESXi, VirtualBox, OpenStack, KVM/QEMU, Docker, AWS, Google Compute Cloud) and automates LPListening Post setup/configuration using a combination of kickstart files and Ansible playbooks.
