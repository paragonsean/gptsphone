Asterisk Service Run Level
The Asterisk server is a physical server in NDBNetwork Devices Branch Lab that runs asterisk, a free call manager for the VOIPVoice over Internet Protocol (Internet telephony) phones.  The asterisk service must be running for the HG VOIPVoice over Internet Protocol (Internet telephony) demo to work.  After a reboot of the Asterisk server, the asterisk service did not start automatically and the HG VOIPVoice over Internet Protocol (Internet telephony) demo did not work.  To configure the asterisk service is set to start up automatically on reboot of the server, type 'chkconfig --list' at the command prompt and locate the asterisk service.  The run level should be set to on for levels 2 - 5.  This has already been set up (change made 2/24/15) for asterisk by typing 'chkconfig asterisk on' at the CLI:
